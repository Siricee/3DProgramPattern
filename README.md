# 《3D编程模式》

本书是开源书，罗列了我从自己的实战项目中提炼出来的关于3D编程（主要包括“3D引擎/游戏引擎”、“编辑器”开发）的各种编程模式

本书作者：杨元超

## 源码索引

要想运行源码，请在Clone本仓库后，先在仓库根目录执行下面的命令：
```js
npm install
npm run bootstrap
```
安装了yarn的同学可以执行下面的命令来替代上面的命令：
```js
yarn
yarn bootstrap
```
